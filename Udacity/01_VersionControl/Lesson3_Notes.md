# Lesson 3

## List of Remotes
```
git remote
```

## Add Origin
```
git remote add origin http....
```

## Verbose
```
git remote -v
```

## Push to GitHub
```
git push origin master
```

## Sync
- Get GitHub repo and local repo back in sync
```
git pull
```

## Update Local Copy
```
git fetch
```

## Merge Branches
```
git merge
git merge master origin/master
```
